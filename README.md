# github-tui
This is a TUI Client for GitHub.

## Features
- issue
  - list/create/close/open
- pr
  - list/create/close/merge
- actions
  - list/log
- file tree
  - preview/open
- project
  - list/open/add/remove/move


## Usage

```sh
# current repository
$ ght

# specified repository
$ ght owner/repo
```

## Installation

```sh
$ git clone https://github.com/skanehira/github-tui
$ go install ./cmd/ght
```
