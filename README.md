# github-tui
This is a TUI Client for GitHub.

## Features
- issue
  - list/create/close/open
- pr
  - list/create/close/merge
- repository
  - list/craete

## Usage

```sh
# repository list
$ ght repo owner

# issue list
$ ght issue owner/repo

# pull request list
$ ght pr owner/repo
```

## Installation

```sh
$ git clone https://github.com/skanehira/ght
$ cd ght && go install
```
